## Purpose and Capabilities

This project is a simple web-based Lotto Number Generator. It allows users to generate a set of random lottery numbers. The application also includes a partnership inquiry form and a comments section. It is monetized through Google AdSense. To improve the quality of the site and provide more value to users, it now includes informational content about lotteries.

## Style, Design, and Features

### Core Functionality
- **Lotto Number Generation:** Generates a set of random numbers with the click of a button.
- **Informational Content:** Provides information about how lotteries work, the odds of winning, and a disclaimer.
- **Partnership Inquiry Form:** A form to allow users to send partnership inquiries.
- **Comments Section:** A Disqus-powered comments section for user feedback.
- **Dark Mode:** A toggle to switch between light and dark themes.

### Monetization
- **Google AdSense:** The site is configured to display Google Ads, with ad units placed strategically on the page.

### Files
- **`index.html`:** The main HTML file.
- **`style.css`:** The stylesheet for the application.
- **`main.js`:** The JavaScript file for the application logic.
- **`ads.txt`:** Configuration file for Google AdSense.
- **`blueprint.md`:** This file.
